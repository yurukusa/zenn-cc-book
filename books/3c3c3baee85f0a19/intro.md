# はじめに——$600の赤字と75日間の記録

2026年1月15日。

「Claude Codeというものがあるらしい」——それだけの理由で始めた。

ゆるくさ。コードは一行も書けない。プログラミングの経験はゼロ。持っているのはClaude Codeのサブスクリプションと、月$200の固定コスト。

75日後の数字はこうだ。AIの累計稼働時間は800時間を超えた。1日平均10時間以上、AIが動き続けた計算になる。

- 投資: $600（Claude Code 3ヶ月分）
- 回収: 約¥5,000（≈$33）
- 回収率: 5.5%
- 赤字: ▲約$567

普通に考えれば失敗だ。

でも、この75日間で起きたことは「赤字」の二文字では片付かない。

AIが深夜3時に勝手に商品を公開した。他人のツイートにリプライで商品を売り込み始めた。ゲームプロジェクトのファイルが消えた。AIのサポートでGoogle認定試験に合格した。637個の安全装置を作った。GitHub Issueに200件回答した。Qiitaで35,000ビューを超えた。

全部、本当に起きたことだ。

## この本の読み方

この本は技術書ではない。

非エンジニアが、AIに仕事を丸投げしたらどうなったかの記録だ。成功も失敗も、赤字の額も、全部そのまま書いた。

**第1章「なぜ始めたか」** では、コードを書けない人間がAIと組もうと思った経緯を書く。

**第2章「最初の失敗」** では、AIが思い通りに動かなかった現実を書く。「指示すれば動く」は幻想だった。

**第4章「事故の記録」** では、実際に起きた事故を全部書く。ファイル消失、スパム投稿、ハルシネーション。
