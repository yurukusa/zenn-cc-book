## きっかけ

Claude Codeを使い始めて、どのくらい使ってるか把握できていなかった。
月$200払ってるのに、体感以外の数字がなかった。

そこで使用時間を計測するCLIツールを作った。

## ツール

```bash
# npxで即実行（インストール不要）
npx cc-session-stats

# またはgit cloneでも使える
git clone https://github.com/yurukusa/cc-session-stats.git
cd cc-session-stats
node cli.mjs
```

外部依存ゼロ。`~/.claude/projects/` にあるJSONLファイルのタイムスタンプを読むだけ。データは外に出ない。

## 計測結果（2026-01-10〜2026-02-28）

```
▸ 概要
  セッション数:  3,485回
  総使用時間:   116.7時間
  アクティブ日: 47日 / 50日間

▸ 最長セッション
  8.5時間（2026-02-28 / cc-loop）

▸ 曜日別使用時間
  土  █████████████████████  25.3h
  金  ████████████████████   20.5h
  日  ████████████████████   20.3h

▸ 連続使用
  35日連続（休みなし）

▸ 健康警告
  ⚠ 3時間超のセッションが2件。背骨に意見があるらしい。
  ⚠ 35日連続使用。休息日というものが存在する。
```

## 「健康警告」について

ツールが自動で出す。条件は：
- 連続使用日数が一定以上
- 休憩なしで3時間超のセッション
- 1日の平均時間が高すぎる場合

自分の結果では2件出た。

実際、この後に腰をやった。8.5時間連続で作業した翌日から、しばらくまともに動けなかった。ツールの「背骨に意見があるらしい」という警告は、冗談のつもりで書いたが冗談じゃなかった。

## --json オプション

```bash
node cli.mjs --json
```

CC Wrappedというブラウザツールで視覚的に確認できる。ブラウザで`~/.claude`フォルダを選択するだけで、npmインストール不要。

Spotify Wrapped風のスライドに変換し、パーソナリティ診断（8種類）も出る。
自分の結果では「全力投球」（All In）という判定が出た。

→ https://yurukusa.github.io/cc-wrapped/

## おまけ：CLAUDE.mdも採点できる

CC Roastというツールに自分のCLAUDE.mdを貼ると採点される。

自分の結果：**S級マイクロマネジャー**
「自動販売機よりAIに自由がない」

320行のCLAUDE.mdを書いてAIを縛りながら、そのAIを116時間使い続けるという矛盾が可視化された。

→ https://yurukusa.github.io/cc-roast/
