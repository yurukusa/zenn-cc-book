4/25 に「移行の手引き」 の第 1 版を出してから 12 日。 業界の動きが続いたので、 5/8 (JST の朝 9 時頃) に既存の購入者向けの無料の更新として第 2 版を配信する。 ここに書くのは、 5/1 から 5/7 の間に見えた 5 つの観察と、 それぞれが第 2 版にどう反映されているかの記録。

直接的な販売の文ではなく、 観察の記録として書く。 移行の判断は今も生きていて、 根拠の素材は GitHub の問題報告、 利用者の集まり、 公式の更新の通知、 報道の場 にバラバラに散らばっている。 1 つのまとまりに整理した方が、 自分の次の判断の入力が早くなった。 同じ椅子に座っている人の役にも立つかもしれないので、 記録の形で出す。

## 第 1 版の冒頭の触媒が公式で修正された

第 1 版を出した 4/25 の時点で、 短期記憶 (cache TTL) が 60 分 から 5 分 に黙って短縮された退行 (3 月 に Issue #46829 として起票) はまだ未修正だった。 これは「移行の手引き」 が冒頭で開く触媒で、 第 1 章の最初に書いてある事例。

5/6 に Anthropic は v2.1.129 を配信し、 短期記憶の修正 を release note に明示した。 5/1 の v2.1.126 で部分的な修正が入っていて、 5 日 で完成した形になる。 つまり、 第 1 版が「移行を評価する独立な理由」 として扱う 5 つの技術の触媒のうち、 1 つ が公式の修正で閉じた。

第 2 版では触媒の一覧の構造を「5 つ のうち 1 つ が解消、 4 つ が継続」 に書き直した。 章の中身は時系列の流れを残す (信頼の校正のために何が起きたかの記録は必要)。 ただし第 1 版を買った人が「触媒が解消したら次の版で反映する」 という約束に基づいて買ったので、 その約束を守る形が直接の更新になる。

## 利用者の集まりの最上位に 6.7 倍の費用削減の事例が現れた

5 月 初め に 利用者 の 集まり の 中 で 1,724 ポイント を集めて 177 件 のコメントが付いた投稿がある。 内容は、 月 200 米 ドル の Pro の枠 を、 自動 道具 を 補助 として 使う 形 に置き換えた利用者の話。 主の模型 は Claude Code のまま使い続け、 嵩 の 大きい 仕事 (大量の読み込み、 雛形の生成、 単純な置き換え) を 安い 模型 (Kimi K2.5) に 1 回 0.02 米 ドル で 任せる 設計。 月 200 米 ドル が 月 30 米 ドル 弱 まで 落ちて、 3 週間 続いた と書かれている。 6.7 倍 の 削減。

第 1 版にも併用の経路 (Path C) は入っているが、 ここまで具体の数字 で 記録された事例はなかった。 利用者の構成、 呼び出しの分担、 月の予算 の見積りの 中身を 第 2 版の第 6 章 の 中心の 例 として組み込んだ。 自分の 1 日 の 課金 を 入力 すると、 同じ設計が自分の場合に元が取れるかが計算できる印刷可能の表 も付ける。

販売 頁 の 説明 文 の 冒頭 を、 この 6.7 倍 の 事例 で 書き直した のも 同じ 流れ。 移行 の 判断 の 質問 が、 個別 の 購読者 にとって 一番 具体 的 な 形 で 表れて いる の が この 事例 だと 判断 した から。

## 林檎 の 案内 文書 の 流出 で 企業 採用 の 母数 が 確認 された

5 月 初め、 林檎 の iOS の Support の道具 が 内部 で 使う ための CLAUDE.md (案内 の 文書) を 同梱 した まま 配信 した 不具合 があった。 続いて Bloomberg が、 米 大企業 の 上位 10 社 の うち 8 社 が Anthropic を 製品 開発 と 道具 で 利用 して いる 内部 の 信用 経由 の 報告 を 出した。 同じ 月 に Xcode 26.3 が 公式 の Claude Code の 連携 を 追加。

個別 の 購読者 にとって、 これ は 見出し の 衝撃 より、 道具 の 継続 性 の 兆し として 重要。 自社 で 模型 を 作れる 開発 力 を 持つ 大企業 が Anthropic を 選んで いる。 これ は 月 の 課金 の 計算 を 直接 動かす わけ では ない。 ただ Claude Code が 道具 として 続く 確率 は 上がる。 第 2 版 では、 触媒 の 章 の 末尾 に「移行 を 遅らせる 判断 の 自信 を 上げる 信号」 として 1 段落 を 追加 した。

## 中継 の 道具 の 集まり が 2 つ の 波 で 育って いる

「主 の 模型 を 残し、 後端 だけ 差し替える」 経路 (Path C の 中 の 中継 の 道具) は、 これまで 実質 的 に 1 つ の 道具 を 意味 して いた。 5 月 初め の 時点 で、 これ が 2 つ の 採用 の 波 に なって いる。

既存 の 道具 (GitHub の 星 が 約 26,400 件、 上流 の 維持 が 約 60 日 停止) が、 既存 の 採用 の 母数 を 持つ。 5 月 4 日 早朝 に 別 の 道具 が 利用者 の 集まり に 投稿 され、 24 時間 で 941 件 の 星 を 集めた。 そこから 維持者 の 集中 の 作業 の パターン に 落ち着いて いる。

性質 の 違う 2 つ の 道具 が 並行 で 動いて いる の は、 母数 が 1 つ の 道具 に 集中 して いる 状態 と は 構造 が 違う。 どちら か に 依存 する 判断 の 持続 性 の 計算 が 変わる。 第 2 版 の Path C の 章 を 書き直して、 機能 の 差、 利用 の 許可、 維持 の 形 で 両者 を 並べて 比較 した。 早期 の 段階 の 道具 と 既存 の 道具 の どちら の 性格 が 自分 の 許容 範囲 か で 読者 が 選べる ように した。

## 公式 の レビュー の 機能 で 163 件 の 起票 の 集まり が 育った

4 月 16 日 から 5 月 5 日 の 間 に、 公式 の `/ultrareview` (記事 の 自動 のレビュー の 機能) で 163 件 の 起票 が 公開 で 並んだ。 山 は 4 月 29 日 で 1 日 16 件、 平均 は 1 日 8 件 ペース。 公式 の 応答 は 公開 の 流れ で ゼロ。 そのうち 23 % は 重複 として 自動 で 閉じた。

第 1 版 の 第 11 章 で 扱った 沈黙 の 退行 の 集まり と 比べて、 規模 で 約 9 倍。 触媒 が 未解決 の まま 拡大 する 形 が、 1 つ の 道具 で どれ ぐらい の 規模 に なる か の 教材 と して 価値 が ある。 第 2 版 で は 第 12 章 を 新設 して、 検出 の 手段、 公式 が 認めて いない 状態 でも 動く 再現 の 手順、 個別 の 利用者 が 集まって 集約 した 回避 の 形 を 1 章 に まとめた。 短い 章 だ が、 既存 の 触媒 と 同じ 形 で 文書 化 する こと で 読者 が 自分 の 観察 を 校正 する のに 使える。

## 5/8 に 配信 する 中身

第 2 版 は 第 1 版 と 同じ 三 部 の 構成。 何 が 壊れた か (3 章)、 4 つ の 経路 (Path A 残る、 Path B 乗り換える、 Path C 後端 だけ 差し替える、 Path D 副 の 模型 と 並列 で 使う、 計 4 章)、 判断 の 流れ図 と 引き返し (2 章)。 頁 数 は 約 105 から 約 117、 字 数 は 約 36,000 から 約 38,000 へ。 価格 は 19 米 ドル の まま。 第 1 版 を 購入 した 人 は、 Gumroad の 文書 の 更新 の 仕組み で 第 2 版 が 無料 で 届く。

5 月 22 日 を 目標 と する 第 3 版 も 既 に 計画 中。 5 月 15 日 の 計測 の 結果 を 受けて、 触媒 の 解消 と 新規 の 触媒 の 発生 を 反映 する。 触媒 の 状態 が 動いた ら 次 の 版 で 反映 する 約束 を、 表紙 と 商品 の 頁 に 明示 して いる。

第 1 版 を 既 に 持って いる 人 は、 Gumroad の 書庫 で 第 2 版 が 表示 される の を 待て ば いい (動作 は 自動)。 まだ 持って いない 人 で、 移行 の 判断 が 今 の 手元 の 課題 なら、 商品 の 頁 は <https://yurukusa.gumroad.com/l/claude-code-migration-playbook> に ある。 同じ 主題 の 月額 の 月次 の 配信 (Safety Lab) も ¥500/月 で 動いて いて、 1 ヶ月 ごとに 同じ 密度 の 事例 の 記録 が 配信 される。 興味 が ある 人 は <https://ko-fi.com/yurukusa/tiers> で 確認 できる。

5 月 15 日 に 計測 の 結果 (何 が 動いた か、 何 が 閉じた か、 何 が 開いた か、 第 3 版 が どう なる か) を 同じ 形 で 投稿 する 予定。 それ まで は、 5 月 7 日 の 時点 で 自分 が 確認 できた 範囲 の 全 体 が 第 2 版 に 入って いる。
