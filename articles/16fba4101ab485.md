cc-safe-setupが310 hooksに到達した。エコシステム全体を俯瞰する。

## 3つのプロダクト

### 1. cc-safe-setup（npmパッケージ）
```bash
npx cc-safe-setup --shield
```
- **310 hooks**（安全ガード、コード品質、セキュリティ、React、a11y）
- **45 CLIコマンド**
- 5言語（Bash, Python, Go, TypeScript, Rust）

### 2. cc-hook-registry（Webレジストリ）
- **210 hooks**をブラウザで検索・フィルタ
- https://yurukusa.github.io/cc-hook-registry/

## 主要コマンド

```bash
npx cc-safe-setup --shield          # 最大安全
npx cc-safe-setup --guard "rule"    # 即時強制
npx cc-safe-setup --suggest         # リスク予測
npx cc-safe-setup --from-claudemd   # ルール変換
npx cc-safe-setup --team            # チーム共有
npx cc-safe-setup --init-project    # 完全セットアップ
npx cc-safe-setup --score           # CIスコア
npx cc-safe-setup --test-hook name  # hookテスト
npx cc-safe-setup --why name        # なぜ存在するか
```
